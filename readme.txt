== Description ==

Embed and display testimonials from [Senja](https://senja.io/) with ease using this WordPress Plugin.

This WordPress plugin allows you to select and display testimonials from Senja using:
- Blocks
- Shortcode
- Elementor Page Builder

Senja is a service that allows you to easily collect text and video testimonials. It loads a small script in the editor for choosing testimonials.

On front, it loads the Senja script on pages where such testimonial widgets are shown.
